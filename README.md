#  Pizza API Challenge

A Flask RESTful API to manage restaurants, pizzas, and their relationships. Built with Flask, SQLAlchemy, Flask-Migrate, and Marshmallow.

---

##  Technologies Used

- Python 3.12+
- Flask
- SQLAlchemy
- Flask-Migrate
- Marshmallow
- SQLite (default DB)
