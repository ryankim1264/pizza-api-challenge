#  Pizza API Challenge

A Flask API to manage restaurants, pizzas, and their relationships. Built with Flask, SQLAlchemy, Flask-Migrate.

---

##  Technologies Used

- Python
- Flask
- SQLAlchemy
- Flask-Migrate
- SQLite (default DB)

---

## Create a virtual Enviroment

pipenv install flask flask_sqlalchemy flask_migrate
pipenv shell
